staticjMod.extend(arg1, arg2, arg3)

F:/Projects/jMod/main/bin/jMod.full.js, line 972
Copy all the properties of one object onto another. If only a single object is given, the properties are copied onto the jMod object.
This version of extend is designed to handle scope and permission issues by cloning the object into the correct scope when an error occurs.
Based on jQuery.extend

arg1 boolean | object (boolean) make deep copy, (object) target object if multiple object arguments exist, (object) object to copy onto jMod object if it is the only input
arg2 object optional Object to copy to target (if arg1 is object) or target object (if arg1 is boolean and arg3 exists) or object to copy onto jMod object
arg3 object optional repeatable Object to copy onto target object
